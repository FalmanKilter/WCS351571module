Тестовое задание для роли Разработчик в отдел Систем управления

1) Для работы над тестовым заданием нужно 
скачать и установить программное обеспечение Matlab
(https://www.mathworks.com/campaigns/products/trials.html)
включающий в себя:
- MATLAB
- Simulink
- Stateflow

2) Необходимо создать программный модуль обрабатывающий сигналы от педали газа.

Педаль газа WCS-351571 от компании Williams получает питание по 2м независимым каналам, и выдаёт уровень напряжения соответствующий углу нажатия по 2м независимым каналам с разной характеристикой. Подробнее в спецификации на оборудование WM-526-Specification-Sheet.pdf

Программный модуль будет получает на входе сигналы от педали:
- Измеренное напряжение питания по 2м каналам
- Уровень напряжения соответствующий углу нажатия по 2м каналам

Необходимо вычислить на выходе:
- Степень нажатия на педаль в процентах от 0% до 100%
- Cигналы об ошибках во входных данных

3) Просьба все модели сохранить в версию Matlab 2017a, без использования тулбоксов из старших версий
